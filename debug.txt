Como executar o debug do Visual Studio
• Para marcar uma linha de breakpoint: Depurar -> Alternar ponto de interrupção (F9) 
• Para iniciar o debug: Depurar -> Iniciar depuração (F5) 
• Para executar uma linha: F10 (ou ir em Depurar > Pular Método)
• Para interromper o debug: SHIFT + F5 (ou clicar no símbolo de "stop")
